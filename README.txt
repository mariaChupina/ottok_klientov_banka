ПРОГНОЗИРОВАНИЕ ОТТОКА КЛИЕНТОВ БАНКА
Задача: спрогнозировать, уйдет клиент из банка в ближайшее время или нет. 
В качестве метрики принята мера F1, а результаты оцениваются по ROC-AUC. Удовлетворяющее заказчика значение F1 должно быть не меньше 0,59.
Данные были предобработаны и подготовлены к исследованию. 
Категориальные столбцы обработаны техникой One-hot-encoding, количественные столбцы масштабированы с помощью StandardScaler.  Данные поделены на train, valid и test в соотношении 60:20:20.
В процессе исследования выявлен дисбаланс классов.
Были построены модели с разными техниками борьбы с дисбалансом. 
с исходным дисбалансом для отслеживания эффективности
дисбаланс ликвидируется за счет применения атрибута class_weight='balanced'
дисбаланс ликвидируется за счет техники upsampling
дисбаланс ликвидируется за счет техники downsampling

Были использованы 3 модели:
логистическая регрессия
решающее дерево
случайный лес

По итогу исследования лучшие результаты дал случайный лес со взвешенными классами: F1 = 0.653, AUC-ROC = 0.869.
Модель прошла проверку на адекватность и показала на тестовых данных следующие значения:
Accuracy -  0.8215
F1 -  0.6115342763873776
ROC-AUC -  0.8576079932012135
